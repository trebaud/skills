# Condensing patterns

Concrete, ordered moves for cutting text. Apply top to bottom — early cuts make later ones obvious.

## 1. Delete filler words

These rarely carry meaning. Delete unless they're doing real work:

- Hedges: *just, really, very, quite, actually, basically, simply, essentially, literally, rather, somewhat, fairly, pretty, kind of, sort of*
- Throat-clearing: *It's worth noting that, It is important to note, Needless to say, As we can see, At the end of the day, When all is said and done, For what it's worth*
- Redundant intensifiers: *very unique → unique, completely eliminate → eliminate, absolutely essential → essential, end result → result, final outcome → outcome, advance planning → planning, past history → history*
- Empty qualifiers: *in some sense, to a certain extent, for all intents and purposes, as a matter of fact*

## 3. Fix weak constructions

- **Expletives** — drop "There is/are" and "It is" openers:
  - *There are many users who complain* → *Many users complain*
  - *It is important that you save* → *Save* / *You must save*
- **Passive → active**: *The file was deleted by the script* → *The script deleted the file*
- **Nominalizations (hidden verbs)**: *make an assumption* → *assume*; *perform a calculation* → *calculate*; *reach a conclusion* → *conclude*; *give an explanation* → *explain*
- **Negatives → positives**: *not many* → *few*; *did not remember* → *forgot*; *not the same* → *different*; *not able* → *unable*
- **Of-phrases → possessives/adjectives**: *the output of the function* → *the function's output*; *a list of errors* → *an error list*

## 4. Cut at the sentence and paragraph level

- **Split convoluted sentences.** One idea per sentence. If a sentence has 3+ clauses or runs past ~25 words, break it.
- **Delete redundant sentences.** If two sentences make the same point, keep the stronger one.
- **Merge when shorter.** Sometimes two short choppy sentences combine into one tighter one — do this when it cuts words.
- **Cut redundant examples/restatements.** "X. In other words, X." → keep one.
- **Front-load the point.** Move the conclusion to the start of the paragraph; cut the wind-up.
- **Prune lists.** Combine overlapping bullets; cut items that repeat each other.

## 5. Trim qualifiers and modifiers

- Cut adjectives/adverbs that don't change meaning: *successfully launched → launched*, *carefully reviewed → reviewed* (unless the manner matters).
- Replace adjective+noun with a precise noun: *a positive improvement → an improvement*, *a quick five minutes → five minutes*.

## What NOT to cut

- Facts, numbers, names, dates, units.
- Qualifiers that change the claim's truth: *usually, only, except, before/after, at least, up to, must/may*.
- Negations and conditions that flip meaning.
- Domain terms the reader needs.
- Code, identifiers, links, and quoted material.

If removing a word changes what the sentence asserts, it was load-bearing — put it back.
